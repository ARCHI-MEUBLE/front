🚀 DÉMARRAGE RAPIDE

  1. Démarrer le serveur:
     → npm run dev

  2. Accéder à la page:
     → http://localhost:3000/stripe-payment

  3. Tester avec une carte Stripe:
     → 4242 4242 4242 4242
     → Date: 12/25 (future)
     → CVC: 123
     → CP: 12345

  4. Voir le résultat:
     → /payment-success (si réussi)
     → /payment-cancel (si annulé)
